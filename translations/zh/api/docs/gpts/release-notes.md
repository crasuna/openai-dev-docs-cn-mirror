# GPT 发布说明

跟踪 OpenAI GPTs 的更新。你也可以查看更广泛的 [ChatGPT 版本发布说明](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)，其中会分享新功能和新能力。本页面会尽力维护，但可能无法反映正在进行的所有变更。

### May 13th, 2024

- Actions 可以在每次请求中[返回](https://developers.openai.com/api/docs/actions/getting-started/returning-files)最多 10 个文件，并将其整合进对话

### April 8th, 2024

- Code Interpreter 创建的文件现在可以被[包含](https://developers.openai.com/api/docs/actions/getting-started/sending-files)在 POST 请求中

### Mar 18th, 2024

- GPT Builders 可以查看并恢复其 GPTs 的先前版本

### Mar 15th, 2024

- POST 请求可以从对话中[包含最多十个文件](https://developers.openai.com/api/docs/actions/getting-started/including-files)（包括 DALL-E 生成的图片）

### Feb 22nd, 2024

- 用户现在可以为 GPTs 评分，这会为 builders 提供反馈，并为 Store 中的其他用户提供信号

- 用户现在可以在 Builders 选择加入时为其留下私密反馈

- 每个 GPT 现在都有 About 页面，其中包含该 GPT 的信息，包括评分、类别、对话数量、入门提示等

- Builders 现在可以将其 Twitter、LinkedIn 和 GitHub 社交资料链接到自己的 GPT

### Jan 10th, 2024

- [GPT Store](https://openai.com/blog/introducing-gpts) 已公开发布，包含类别和各种排行榜

### Nov 6th, 2023

- [GPTs](https://openai.com/blog/introducing-gpts) 允许用户针对各种用例自定义 ChatGPT，并与其他用户分享
