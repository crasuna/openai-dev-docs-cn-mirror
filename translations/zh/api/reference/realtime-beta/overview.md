# Realtime Beta 概览

通过 WebRTC、WebSocket 和 SIP 等低延迟接口，与多模态模型实时通信。原生支持语音到语音，也支持文本、图像和音频输入与输出。
[进一步了解 Realtime API](https://developers.openai.com/docs/guides/realtime)。
