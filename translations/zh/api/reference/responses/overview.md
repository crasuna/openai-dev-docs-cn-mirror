# Responses 概览

OpenAI 用于生成模型响应的最先进接口。支持文本和图像输入，以及文本输出。你可以使用先前响应的输出作为输入，与模型创建有状态交互。通过用于文件搜索、web 搜索、computer use 等能力的内置工具扩展模型能力。还可以通过 function calling 让模型访问外部系统和数据。
