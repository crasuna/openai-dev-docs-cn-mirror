# 功能成熟度

有些 Codex 功能会带有成熟度标签发布，帮助你理解每项功能的可靠程度、可能发生的变化，以及可以期待的支持级别。

| 成熟度            | 含义                                                                                  | 指引                                                                  |
| ----------------- | ------------------------------------------------------------------------------------- | --------------------------------------------------------------------- |
| Under development | 尚未准备好使用。                                                                      | 不要使用。                                                            |
| Experimental      | 不稳定，OpenAI 可能移除或更改它。                                                     | 自行承担风险使用。                                                    |
| Beta              | 已准备好进行广泛测试；大多数方面已完整，但部分内容可能根据用户反馈变化。              | 适合大多数评估和试点；预期会有小幅变化。                              |
| Stable            | 完全受支持、已有文档，并准备好广泛使用；行为和配置会随时间保持一致。                  | 可安全用于生产环境；移除通常会经过弃用流程。                          |
